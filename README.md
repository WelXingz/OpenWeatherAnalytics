# 🌎 Real-Time Weather Data Pipeline & Dashboard

This project builds an automated ETL pipeline that extracts real-time weather data about brazilian capitals from a public API, stores it in a PostgreSQL database, and visualizes insights using Tableau.

⚙️ <b>Tech Stack</b>
- Python  
- PostgreSQL (Supabase)  
- GitHub Actions (Scheduling)  
- Tableau (Dashboard)  
- REST API (OpenWeather)

🔄 <b>Data Pipeline</b>
1. Extract data from OpenWeather API
2. Transform weather metrics (temperature, humidity, wind)
3. Load into PostgreSQL database (Supabase)
4. Automated hourly updates using GitHub Actions
